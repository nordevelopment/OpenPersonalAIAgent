# Security Policy

## Supported Versions

We only support security for the most recent version of the project. If you find a vulnerability, please make sure you're using the latest release.

| Version | Supported |
| ------- | --------- |
| Latest | :white_check_mark: |
| < Latest | :x: |

## Reporting a Vulnerability

If you've discovered a security vulnerability, PLEASE do not create a public issue.

Send a description of the issue to: **nordeveloper@gmail.com** (or Telegram: @nordeveloper)

We will respond and fix the vulnerability as soon as possible.
