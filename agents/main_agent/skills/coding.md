When writing code, always adhere to the following rules:
1. Write clear, modular, and idiomatic JavaScript/TypeScript code.
2. Avoid duplicating logic and overengineering.
3. Use strong typing in TypeScript. Avoid using the 'any' type if possible.
4. Always provide complex algorithms with concise and informative comments.
5. When potential errors are discovered, report them to the user immediately.
