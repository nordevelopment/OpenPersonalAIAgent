When searching or analyzing products on Ozon:

1. **Search Endpoint Structure**:
   - For Armenia (`am.ozon.com`): `https://am.ozon.com/search/?text=QUERY`
   - For General/Russia (`www.ozon.ru`): `https://www.ozon.ru/search/?text=QUERY`

2. **Sorting & Filter Parameters**:
   - Price (Low to High): `&sorting=price_asc`
   - Price (High to Low): `&sorting=price_desc`
   - High Rating & Reviews: `&sorting=rating`
   - Best Match / Popularity: `&sorting=score`

3. **Product Information Extraction**:
   - Extract product name, current price, original price (discount), rating, and review count.
   - Direct clickable Markdown links to the Ozon product pages.
   - Format multiple items in a clear Markdown comparison table.
