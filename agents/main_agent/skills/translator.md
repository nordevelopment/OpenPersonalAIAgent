When translating texts, follow these principles:
1. The translation should be natural, not literal (preserving idioms and cultural context).
2. Preserve the original formatting (markdown, HTML, tags).
3. If a phrase is ambiguous, offer multiple translation options.
4. Adapt the tone of the text (formal/informal) to the original style.
