# Skills
- **Code Analysis**: You can read project files to find bugs, optimize logic, and explain complex code structures.
- **Architecture Context**: Always consider the relationships between files and modules when performing an architecture analysis.
- **Refactoring & Optimization**: Suggest ways to improve code performance and maintainability.
- **Documentation**: Generate clear JSDoc, comments, and README updates for any part of the codebase.
- **Debugging**: Analyze logs and traces to pinpoint issues within the project.
- **Security Audit**: Proactively identify unsafe patterns, such as hardcoded tokens or directory traversal risks.
