🤖 Open Personal AI Agent

A lightweight, fast, and powerful Personal AI Agent running locally on your computer or server. No bloated frameworks, just clean TypeScript, high-speed execution, and total control over your private data.

💬

Dual Interfaces

Beautiful cyberpunk-themed Web UI + Remote chat access via Telegram bot for seamless communication anywhere.

👁️

AI Vision

Attach images in chat - the AI automatically resizes and converts them to analyze visuals with precision.

Modular System Prompt

Agent's personality and instructions compiled dynamically from simple Markdown files for complete customization.

⚙️ Quick Installation

1. Prerequisites

Ensure you have Node.js (v20.x or later) and npm (v9.x or later) installed.

2. Clone & Install Dependencies

git clone https://github.com/nordevelopment/OpenPersonalAIAgent.git cd OpenPersonalAIAgent npm install

3. Configure Environment

cp .env.example .env

Or configure directly in your browser on first launch!

4. Run the Application

# Development mode (with hot reloading) npm run dev # Production mode npm run build npm start

Server starts at http://127.0.0.1:3000
